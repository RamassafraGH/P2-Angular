# P2-Angular

Repositorio para el proyecto básico de Angular.

## 🚀 Descripción
Este proyecto es una aplicación Angular creada con fines de aprendizaje.  
Incluye ejemplos de:
- Uso de **standalone components**.
- Configuración de **rutas** (`home`, `ventas`, `sucursales`, `clientes`).
- Componentes personalizados como un **reloj en el header**.
- Estilos modernos con CSS variables y diseño responsivo.

## 📂 Estructura del proyecto
- `src/` → Código fuente principal.
- `app.ts` → Componente raíz de la aplicación.
- `routes.ts` → Configuración de rutas.
- `footer/` → Componente de pie de página.
- `reloj/` → Componente de reloj en tiempo real.
- `public/` → Archivos estáticos.
